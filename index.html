Multi robot simulator unity

This is a solo project that has the goal of using Unity as a testing environment for robots. To do this, I needed to port a prebuilt robot chassis (Leo Rover) over to a Unity-supported file format, from URDF. Then, as solutions for bridging unity to ros are clunky and slow, we use ros2 for the unity package, which allows us to implement ros2 with C\#. This allows us to manipulate and control the robot with the Ros Unity Bridge. Then, on top of this, we add functionality for autonomous navigation by porting over muiltCol-slam to ros2 as well as April Tag for commands for task performance identification and location.
